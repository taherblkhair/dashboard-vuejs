# my-vue-app

Minimal Vue 3 + Vite project scaffold with Tailwind CSS and a responsive layout (Navbar + Sidebar).

How to run

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

Create GitHub repository and push

1. Create a remote repository on GitHub (choose a name, e.g. `my-vue-app`).
2. Link and push from your local machine (example using HTTPS):

```bash
git remote add origin https://github.com/<your-username>/my-vue-app.git
git branch -M main
git push -u origin main
```

Or with the GitHub CLI:

```bash
gh repo create <your-username>/my-vue-app --public --source=. --remote=origin --push
```

Notes
- Make sure you have an account and proper auth (SSH key or gh login) before pushing.
- Adjust README and remove these notes before publishing.
